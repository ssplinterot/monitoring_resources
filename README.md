# Сетевая система мониторинга ресурсов узлов

Курсовой проект, БГУИР 2026. Архитектура **агент → сервер**: агент собирает метрики машины и отправляет на сервер по TCP, сервер отображает всё в GUI.

## Что умеет
- Мониторинг CPU, RAM, Disk в реальном времени
- Топ процессов по CPU и RAM (видно какое приложение нагружает систему)
- Графики динамики нагрузки
- Предупреждения при превышении пороговых значений
- Лог метрик в CSV
- Автопереподключение агента при обрыве связи

## Стек
Python 3 · PyQt5 · pyqtgraph · psutil · TCP/IP · JSON

## Запуск

```bash
pip install PyQt5 pyqtgraph psutil
python3 server.py   # сначала сервер
python3 agent.py    # потом агент
```

При первом запуске агент создаст `config.ini` — там можно поменять IP сервера и имя узла.
